# 모바일 설치 · 백업 가이드

## 지금 기준으로 추천 사용 방식
- 아이폰에서 홈 화면에 추가해 전용기기처럼 사용합니다.
- 작성 내용은 기기 안에 저장됩니다.
- 다른 기기로 옮길 때는 `전체 데이터 백업` 파일을 내보내 전달합니다.
- 새 기기에서는 `백업 불러오기`로 그대로 복원합니다.

## 모바일에서 할 수 있는 것
1. 홈 화면에 추가해서 앱처럼 실행
2. 네트워크가 없어도 기존에 열어둔 앱으로 계속 작성
3. 기기 안에 자동 저장
4. 전체 데이터 백업 파일 저장
5. AirDrop, 파일 앱, 메신저 등으로 백업 파일 전달
6. 다른 아이폰, 아이패드, PC 브라우저에서 복원

## 실행 방법
```bash
cd /Users/sonjisun/자동화/모임총무
python3 app_server.py
```

기본 주소:
- 로컬: `http://127.0.0.1:4174`
- 같은 와이파이: `http://내-PC-IP:4174`

## 아이폰 설치 방법
1. Safari에서 주소 열기
2. 하단 공유 버튼 누르기
3. `홈 화면에 추가` 선택
4. 홈 화면의 `모임 총무 보드` 아이콘으로 실행

## 기기 이전 방법
1. 기존 기기 `설정 > 관리 도구 > 전체 데이터 백업`
2. 필요하면 `백업 공유`로 파일 전달
3. 새 기기에서 앱 열기
4. `설정 > 관리 도구 > 백업 불러오기`
5. 기존 데이터 그대로 복원 확인
